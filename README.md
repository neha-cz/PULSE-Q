# PULSE-Q

Preliminary simulation framework for weak coherent pulse (WCP) quantum key distribution on a 12U CubeSat.

This project explores:

- Poisson photon-number statistics for phase-randomized WCPs  
- Truncated Fock-space modeling (QuTiP)  
- Polarization encoding for BB84  
- Basic loss, dephasing, and stray-light effects  

## Requirements

- Python 3.x  
- NumPy  
- SciPy  
- Matplotlib  
- QuTiP  
- Jupyter Notebook
